# Ebb Platform Server
Server to provide cloud features for Ebb. This includes things like gamification, friends, and backups

To install dependencies:

```bash
bun install
```

To run:

```bash
bun dev
```

## Getting Started
- copy .env.example to .env
- get values for .env files (supabase for example)
